# Руководство пользователя «Цифровой двойник»

## Введение

**«Цифровой двойник»** — это симулятор работы карьера, созданный для математического моделирования технологических процессов. Он позволяет создать виртуальную копию карьера и отвечать на ключевые вопросы: «Что, если…?» и «Как лучше сделать…?».

### Функциональные возможности:
- Получение объективных данных для принятия решений
- Расчет теоретически возможного объема добычи
- Сравнение эффективности ручного и автоматического планирования
- Тестирование гипотез без риска
- Моделирование нештатных ситуаций (поломки, закрытие маршрутов)
- Автоматический расчет оптимальных маршрутов техники
- Ведение детального лога событий

## Основные разделы системы

### 1. Сценарии
Раздел для создания, настройки и запуска виртуальных экспериментов.

**Основные функции:**
- Создание и редактирование сценариев
- Установка дат и времени симуляции
- Выбор карьера из системы
- Режимы расстановки техники:
  - **Автоматический** - используются все объекты из справочника
  - **Ручной** - выбор конкретной техники
- Учет достоверности (создание трех вариантов модели):
  - Идеальный (без сбоев)
  - Пессимистичный (со случайными поломками)
  - Достоверный (прогноз на основе нескольких симуляций)

### 2. Результаты моделирования
Инструмент для визуализации и мониторинга данных симуляции.

**Отображаемая информация:**
- Информация об объектах моделирования
- Графики ключевых показателей:
  - Динамика топлива по часам
  - Интенсивность перевозок
- Детальная информация по технике:
  - Экскаваторы: количество, идентификаторы, масса, рейсы
  - Самосвалы: количество, идентификаторы, масса, рейсы
  - Пункты разгрузки: количество, идентификаторы, масса, рейсы
- Хронология событий
- Детализированные таблицы рейсов

### 3. Справочники
База данных всех объектов модели с предустановленными характеристиками.

**Настраиваемые объекты:**
- **Карьеры**: название, координаты, настройки времени, графики смен
- **Самосвалы**: модель, характеристики, координаты, надежность
- **Экскаваторы**: модель, расположение, производительность, статистика отказов
- **Площадки разгрузки**: координаты, вместимость, угол наклона
- **Заправки**: расположение, количество пистолетов, скорость заправки
- **Маршруты**: начальная и конечная точки, скорости движения, уклон

### 4. Редактор карты
Инструмент для визуального проектирования инфраструктуры карьера.

**Возможности:**
- Загрузка горных планов в формате DXF
- Размещение объектов на карте
- Построение графов дорог
- Визуальное редактирование элементов модели
- Настройка привязки к дорожной сети

## Создание и запуск модели

### Шаг 1. Подготовка данных в «Справочниках»

#### 1.1. Настройка карьера
- Выбор или создание проекта карьера
- Задание характеристик:
  - План эксплуатации
  - Название
  - Географические координаты
  - Настройки времени (часовой пояс)
  - График перерывов и смен

#### 1.2. Создание шаблонов объектов
**Шаблоны позволяют сохранить параметры для быстрого добавления:**

- **Самосвалы**: модель, координаты, показатели надежности
- **Экскаваторы**: модель, расположение, производительность, статистика отказов
- **Площадки разгрузки**: вместимость, угол наклона, количество машин
- **Заправки**: расположение, количество пистолетов, скорость заправки

#### 1.3. Добавление маршрутов
- Указание начальной и конечной точек
- Задание параметров: скорости движения, уклон

#### 1.4. Расписание плановых простоев
- Выбор даты и смены
- Добавление интервалов простоя для самосвалов
- Сохранение в системе

#### 1.5. Расписание взрывных работ
- Указание даты и времени взрывных работ
- Определение зоны работ на карте с помощью полигонов

### Шаг 2. Подготовка в «Редакторе карт»

#### 2.1. Создание дорожной сети
- Включение режима редактирования
- Построение маршрутов с помощью полилиний
- Сохранение изменений

#### 2.2. Привязка объектов к дорожной сети
- Перемещение точек маршрутов и иконок техники
- Связывание участков дорог с объектами
- Проверка непрерывности маршрутов

### Шаг 3. Запуск моделирования в «Сценариях»

#### 3.1. Создание сценария
- Ввод уникального названия
- Установка периода моделирования
- Выбор карьера

#### 3.2. Настройка параметров
- Выбор режима расстановки техники
- Настройка учета достоверности
- Ручное распределение самосвалов по маршрутам (при необходимости)

#### 3.3. Запуск симуляции
- Сохранение настроек
- Запуск расчета модели

### Шаг 4. Просмотр результатов

#### 4.1. Анализ результатов
- Просмотр временного периода симуляции
- Оценка показателей производительности
- Изучение сводки по технике и площадкам

#### 4.2. Графики и отчеты
- Графики вывезенной массы и количества рейсов
- Производительность объектов
- Отслеживание работы самосвалов в реальном времени
- Просмотр хронологической ленты событий

## Глоссарий терминов

| Термин | Описание |
|--------|-----------|
| **Карьер** | Основной проект в системе, объединяющий все объекты и настройки |
| **Самосвал** | Единица транспорта для перевозки породы |
| **Экскаватор** | Единица техники для погрузки породы в самосвалы |
| **Площадка разгрузки** | Объект, где самосвалы разгружают породу |
| **Заправка** | Элемент инфраструктуры для заправки техники |
| **Маршрут** | Путь между двумя точками карьера с заданными параметрами |
| **Граф дорог** | Цифровая модель дорожной сети карьера |
| **Автоматический режим** | Система автоматически выбирает технику из Справочников |
| **Ручной режим** | Пользователь вручную выбирает технику для симуляции |
| **Учёт достоверности** | Параметр, определяющий учёт случайностей и отказов техники |
| **Шаблоны** | Предустановленные наборы параметров для быстрого добавления объектов |
